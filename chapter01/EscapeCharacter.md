| 符号 |         含义         |
| :--: | :------------------: |
|  \t  | 一个制表位，实现对齐 |
|  \n  |         换行         |
| \ \  |        一个\         |
| \ "  |        一个"         |
| \ '  |        一个'         |
|  \r  |       一个回车       |

练习：

``` java
public class ChangeChar {
	//只使用一句输出语句
	public static void main(String[] args) {
		System.out.println("书名\t作者\t价格\t销量\n三国\t罗贯中\t120\t10000");
	}
}
```
